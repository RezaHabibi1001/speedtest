# Fast Style Download Test

یک پروژه سبک و تک‌فایلی شبیه fast.com برای اندازه‌گیری **سرعت دانلود** با استفاده از HTML، CSS و JavaScript خالص.

## ویژگی‌ها

- اندازه‌گیری فقط سرعت دانلود.
- رابط کاربری ساده و مینیمال شبیه fast.com.
- نمایش سرعت لحظه‌ای بر حسب Mbps.
- نمایش بیشترین سرعت، میانگین نهایی و مدت زمان تست.
- دارای دکمه شروع تست و ریست.
- پشتیبانی از لایت مود و دارک مود.
- لایت مود به صورت پیش‌فرض فعال است.
- طراحی فشرده‌تر برای نمایش بهتر روی لپ‌تاپ ۱۴ اینچ بدون اسکرول.

## ساختار پروژه

```text
speed-test/
├── fast-style-download-test.html
└── README.md
```

## نحوه اجرا

1. فایل `fast-style-download-test.html` را داخل مرورگر باز کنید.
2. روی دکمه `شروع تست` کلیک کنید.
3. منتظر بمانید تا تست کامل شود و مقدار نهایی نمایش داده شود.

## نحوه کار

این پروژه با دانلود همزمان داده از یک endpoint تست، حجم داده دریافت‌شده را در بازه زمانی مشخص اندازه‌گیری می‌کند و سرعت دانلود را به Mbps نمایش می‌دهد.

فرمول تقریبی محاسبه:

```text
Mbps = (bytesLoaded * 8) / elapsedSeconds / 1000 / 1000
```

## تنظیمات مهم

مقادیر اصلی داخل فایل JavaScript:

```js
const TEST_URL = "https://speed.cloudflare.com/__down?bytes=25000000";
const TEST_DURATION_MS = 8000;
const UPDATE_INTERVAL_MS = 250;
const MAX_PROGRESS_SPEED = 200;
```

### توضیح تنظیمات

- `TEST_URL`: آدرس فایل یا endpoint تست دانلود.
- `TEST_DURATION_MS`: مدت زمان تست.
- `UPDATE_INTERVAL_MS`: فاصله بروزرسانی رابط کاربری.
- `MAX_PROGRESS_SPEED`: سقف نمایش progress دایره‌ای.

## پیشنهاد برای production

برای دقت بهتر در محیط واقعی:

- از فایل تست بزرگ روی CDN یا سرور خودتان استفاده شود.
- از URLهایی استفاده شود که مشکل CORS نداشته باشند.
- برای جلوگیری از cache مرورگر، از query string تصادفی استفاده شود.
- در صورت نیاز، شروع تست به‌صورت خودکار بعد از load صفحه انجام شود.

## سفارشی‌سازی

### تغییر پیش‌فرض تم

در فایل JavaScript، مقدار زیر تعیین می‌کند که سایت با چه تمی شروع شود:

```js
let currentTheme = "light";
```

### تغییر اندازه کلی layout

برای کوچک‌تر یا بزرگ‌تر کردن صفحه، این بخش‌ها مهم هستند:

- عرض کلی `.app`
- اندازه دایره `.speed-shell`
- فاصله‌های `.panel`
- اندازه فونت `.speed-value`
